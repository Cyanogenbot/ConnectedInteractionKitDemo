# Demo of Arduino Nano RP2040 Connect in CircuitPython with OOCSI and Adafruit LED ring module

<br />

## Hardware

* Arduino Nano RP2040 Connect
* Adafruit LED ring module (12 leds)

<br />

## Wiring

| Adafruit LED ring  |  Arduino Nano RP2040 Connect |
| --- | --- |
| Data Input | D2 pin |
| 5V DC Power | Vin |
| GND | GND |
